# Prework Study Guide Webpage

## Description

This Prework Study Guide was created for boot camp students from the Software Development Bootcamp at the University of Oregon who are going through the prework. It contains notes on HTML, CSS, Git, and JavaScript.

## Installation

N/A

## Usage

To utilize this Prework Study Guide effectively, you should review the notes provided in each section. For guidance on where to begin, please open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows). This action will open a console panel either below or beside the webpage within your browser. Within this panel, you will find a comprehensive list of topics covered in the prework, accompanied by a suggested starting point for your studies.
